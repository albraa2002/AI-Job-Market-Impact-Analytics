🤖 AI Boom & Tech Job Market Impact Analytics

📌 Project Overview
This project visualizes the paradigm shift in the global tech job market driven by the Generative AI boom. Analyzing a simulated dataset of 12,000 job postings from 2023 to 2026, this dashboard contrasts the demand, salary premiums, and work models of 'AI-Augmented/Data Roles' versus 'Traditional Tech Roles'.

🚀 Key Business Insights
The AI Salary Premium: Quantified a significant salary gap, with AI and Data roles commanding notably higher average compensation (The AI Premium) compared to traditional tech positions.
Demand Crossover: Visualized the exponential growth trajectory of AI-related job postings post-mid-2023, effectively overtaking stagnating traditional roles.
Remote Work Dominance: Identified that the vast majority of emerging AI/Data roles offer remote work models, reshaping traditional workplace dynamics.
Modern Tech UI: Designed a premium, Cyberpunk-inspired HTML dashboard with a glassmorphism aesthetic to effectively communicate the futuristic nature of the data.

🛠️ Tech Stack
Language: Python 3.x
Data Manipulation: Pandas, NumPy
Data Visualization: Plotly (Interactive Line, Box/Bar, and Donut charts)
UI/UX: Standalone pure HTML/CSS Grid Dashboard (Neon/Dark Mode)
